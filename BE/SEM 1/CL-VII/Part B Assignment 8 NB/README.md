# Instructions For UBUNTU
1. Make sure you have weka.jar and iris dataset in same directory as your code 
2. To Complie 
```
javac -cp weka.jar MyNaiveBayes.java
```
3. To Run 
```
java -cp weka.jar:. MyNaiveBayes
```
Just replace the ':' with ';' in windows

Reference : http://www.emaraic.com/blog/weka-java-example
